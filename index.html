🌐 Share on Local Network

Let anyone on your office WiFi view your data in read-only mode — no editing, no cloud, no account.

Step 1 — Find your IP address

Mac

Quick way (Terminal):

ipconfig getifaddr en0

Without Terminal:

System Settings → WiFi → click your network name → Details → IP Address

Windows

Quick way (Command Prompt):

ipconfig

Look for "IPv4 Address" under your WiFi adapter

Without Command Prompt:

Settings → Network & Internet → WiFi → your network name → Properties → scroll to IPv4 address

Step 2 — Start sharing (one command)

Open Terminal (Mac) or Command Prompt (Windows) in the folder where index.html lives, then run:

python3 -m http.server 8080

Windows fallback if that doesn't work: python -m http.server 8080

Step 3 — Share the link

Tell the other person to open their browser and go to:

http://[your-ip-address]:8080/index.html

Example: http://192.168.1.42:8080/index.html

How this works: Your computer serves the file. Anyone on the same WiFi sees your live data. They can browse everything but can't edit anything — their browser has no write access to your data. Close the terminal window to stop sharing.
